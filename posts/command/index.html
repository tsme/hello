docker服务

  • 启动docker服务

    systemctl start/stop docker

  • 查看docker状态

    systemctl status docker

  • 设置docker开机自启

    systemctl enable/disable docker

  • 重启docker服务

    systemctl restart docker

  • 查看docker信息

    docker info

容器

  • 查看docker运行中的容器

    docker ps

  • 查看docker所有容器

    docker ps -a

  • 查看容器日志

    docker container logs xxx(容器id/名称)

  • 持续查看容器日志

    docker logs -f name

  • 启动/停止容器

    docker container start/stop/restart xxx(容器id/名称)

  • 删除容器

    docker container rm xxx(容器id/名称)

制作jdk镜像

  • 创建Dockerfile文件 ，vi Dockerfile

    FROM centos:7 ADD jdk-8u181-linux-x64.tar.gz /usr/local/docker/jdk8 ENV JAVA_HOME=/usr/local/docker/jdk8/jdk1.8.0_181
    PATH=/usr/local/docker/jdk8/jdk1.8.0_181/bin:$PATH CMD [‘bash’]

  • 创建jdk镜像文件

    docker build -t jdk:8 . #不要丢掉这里的点,-t表示镜像标识(镜像名),是tag单词的缩写.

  • 运行jdk镜像文件

    docker run -it jdk:8 bash

    java –version

启动jar

docker run -d -p 8080:8080 -v /usr/local/docker/jar/springweb.jar:/var/lib/docker/jar/gateway.jar –name springweb jdk:8 java -jar /var/lib/docker/jar/gateway.jar
